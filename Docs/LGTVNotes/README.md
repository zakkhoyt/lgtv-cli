# LGTV Notes

Quick references for configuring and automating LG OLED displays.

## Document Guide

- [LGOLED.md](LGOLED.md) – Factory-reset checklist covering energy settings, status LEDs, mute menu tweaks, and picture-mode/HGiG pointers.
- [LGOLED_SETUP.md](LGOLED_SETUP.md) – Hardware/software stack for using an LG C1 as a Mac monitor, including LGWebOSRemote, Hammerspoon, MonitorControl, and HDMI audio workarounds.
- [LGOLED_MONITOR.md](LGOLED_MONITOR.md) – Deep dive on day-to-day usage: hidden menus (VRR, hotel mode, Quick Access), service remote terms, macOS external display tips, and brightness/volume tooling.
- [HAMMERSPOON.md](HAMMERSPOON.md) – Notes on diagnosing Hammerspoon hangs with relevant open-file handles and log locations.
- [BETTER_DISPLAY.md](BETTER_DISPLAY.md) – Installation and usage of BetterDisplay CLI/app binaries for querying or setting LG TV brightness from macOS.
