# ``LGTVWebOSController``

A Swift library for controlling LG webOS TVs over WebSocket connections.

## Overview

LGTVWebOSController is a Swift package that provides a complete implementation for controlling LG webOS televisions. It uses WebSocket connections to communicate with the TV, supporting both secure (WSS) and insecure (WS) connections.
